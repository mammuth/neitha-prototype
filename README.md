# NEITHA - antitheft device for women handbags

This is a prototype of a hardware device which prevents theft of women handbags.

## Tech Stack
**Frontend**
- React
- Mobx
- Less
- Webpack

**Backend**
- Python3
- Flask

**Hardware**
- Python2
